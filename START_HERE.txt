================================================================================
  📱 TESTING ON MOBILE (Same Wi-Fi Network)
================================================================================

1. Find your PC's IP address:
   - Open Command Prompt
   - Run: ipconfig
   - Look for "IPv4 Address" (e.g., 192.168.100.9)

2. Start server for network access:
   - venv\Scripts\activate
   - python manage.py runserver 0.0.0.0:8000

3. On your mobile device, open:
   - http://192.168.100.9:8000/home/
   - (Replace with your actual IP address)

================================================================================
  ❓ TROUBLESHOOTING
================================================================================

Problem: "Port already in use"
Solution: python manage.py runserver 8080

Problem: "Module not found"
Solution: pip install -r requirements.txt

Problem: "Database error"
Solution: python manage.py migrate

Problem: "Static files not loading"
Solution: python manage.py collectstatic --noinput

Problem: "Can't access on mobile"
Solution:
  1. Check firewall settings
  2. Make sure both devices are on same Wi-Fi
  3. Use: python manage.py runserver 0.0.0.0:8000
